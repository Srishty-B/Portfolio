• There is a desperate need in animation, gaming and robotics industry, for a tool to animate physical movements to an animated 3D model, at low cost and less hardware. This not only helps in a better and easier method to animate movements, but the low cost associated with it helps different industries to save costs as well.

  • This project aims at creating an application that will allow users to track the movement of their hands in real time.

  • The application will track the motion of hand and display a 3D animated hand on the screen in real time.



  • The application can be updated with other features, like color change options,adding some games that are played using hand gestures etc.

  • The application can be used to record the animation and store it in memory for other uses

  • This project can be successfully scoped up for full body animation in future as well
